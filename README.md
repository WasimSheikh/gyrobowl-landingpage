# Gyro Bowl Landing Page

A modern, responsive landing page for Gyro Bowl restaurant's upcoming launch.

## Features

- Responsive design that works on all devices
- Countdown timer to launch date
- Email notification signup form
- Beautiful hero image with overlay
- Modern typography using Google Fonts
- Smooth animations and transitions
- Social media links
- Mobile-friendly layout

## Setup

1. Clone this repository
2. Open `index.html` in your browser
3. To deploy, simply upload all files to your web hosting service

## Structure

```
gyrobowl-landing-page/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
└── images/
    └── hero.jpg
```

## Customization

- To change the launch date, modify the `launchDate` variable in `js/main.js`
- To update colors, modify the CSS variables in `css/style.css`
- To change the hero image, replace `images/hero.jpg`
